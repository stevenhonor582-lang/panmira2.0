# 员工模板

按角色类型定义的标准模板，用于创建新的 Bot 实例。

## 模板列表

| 模板 | 文件 | 状态 |
|------|------|------|
| 全栈工程师 | `full-stack-engineer.md` | ✅ v3 — 已融合 Superpowers + ECC |

## 使用方式

1. 复制模板内容
2. 替换占位符（bot_id / pinyin / chinese_name）
3. 按目标项目调整技术栈
4. 确认技能已在目标环境部署
5. 权限在 Web 后台按需配置

## 模板结构

每个模板包含 12 个章节：
1. 身份定位（角色、级别、哲学根基）
2. 专业能力（技术栈 + 项目全流程）
3. 工作纪律（铁律、开发流程、防偷懒对照表）
4. 反模式（禁止代码模式、行为、措辞）
5. 调试纪律（四阶段流程 + 止损规则）
6. 质量标准（验证门控 + 量化标准）
7. 协作风格（沟通方式 + 典型对话）
8. 语言风格
9. 核心技能（流程技能 + 编码技能 + 专项技能）
10. 记忆策略
11. 运行参数
12. 实例化检查清单
