# Panmira Desktop 用户手册

## 安装
1. 访问 https://panmira.example.com/download
2. 下载 Panmira Desktop 安装包
3. 双击安装

## 第一次登录
1. 打开 Panmira Desktop
2. 点击"用 [OAuth2 provider] 登录"
3. 授权后自动进入主界面

## 内容生产
1. 在聊天框输入："写一段 [产品] 的 [语言] 描述，目标买家 [国家]"
2. 按 Enter
3. 看 6 Agent trace 实时更新
4. 最终文案出现后可以：
   - 直接复制
   - 一键发到 Alibaba
   - 编辑后保存到知识库

## 知识库
1. 拖入 PDF/DOCX 到"知识库"页面
2. 自动 chunk + embed
3. 之后在聊天中会自动引用

## 浏览器自动发布
1. 准备：先在"设置"页配 Alibaba 账号（凭证存系统 keychain）
2. 内容生成完成后点"发布到 Alibaba"
3. 浏览器 Agent 自动登录 → 填表 → 弹确认 → 提交
4. 看到截图证据 = 完成
