# 商业计划模板

> 适用 topic_category: 商业计划
> 适用 audience: 投资人 / 管理层
> 默认 length: 3000字-5000字+

## 必含章节

1. 执行摘要 (Executive Summary)
2. 市场机会 (Market Opportunity)
3. 解决方案 (Solution)
4. 商业模式 (Business Model)
5. 竞争优势 (Competitive Advantage)
6. 团队 (Team)
7. 财务预测 (Financial Projections)
8. 融资需求 (Funding Requirements)

## 章节写作要点

### 1. 执行摘要
- 1 段话讲清楚"做什么 / 给谁 / 怎么赚钱 / 要多少钱"
- 突出差异化与增长性
- 投资人 30 秒可读完

### 2. 市场机会
- TAM / SAM / SOM（可标注数据来源）
- 趋势驱动因素
- 时间窗口

### 3. 解决方案
- 产品/服务概述
- 解决的核心痛点
- 价值量化

### 4. 商业模式
- 收入来源（订阅 / 交易 / 广告 / 授权）
- 定价策略与依据
- 单位经济学（LTV / CAC / 毛利）

### 5. 竞争优势
- 护城河（技术 / 品牌 / 规模 / 渠道）
- 关键里程碑（已验证的）
- 与竞品的差异化

### 6. 团队
- 核心成员背景（突出相关经验）
- 关键能力互补
- 顾问与董事会

### 7. 财务预测
- 3 年收入预测（关键假设）
- 关键成本结构
- 盈亏平衡时间点
- 情景分析（乐观 / 基准 / 保守）

### 8. 融资需求
- 本轮融资金额
- 资金用途（人员 / 营销 / 研发）
- 18-24 个月 runway
- 期望的合作（不只是钱）
