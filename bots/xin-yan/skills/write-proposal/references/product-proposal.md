# 产品介绍模板

> 适用 topic_category: 产品介绍
> 适用 audience: 客户 / 潜在用户 / 合作伙伴
> 默认 length: 1500字-3000字

## 必含章节

1. 产品概述 (Product Overview)
2. 核心价值 (Core Value Proposition)
3. 目标用户 (Target Users)
4. 关键功能 (Key Features)
5. 差异化优势 (Differentiators)
6. 使用场景 (Use Cases)
7. 定价与获取 (Pricing & Access)
8. 路线图 (Roadmap)

## 章节写作要点

### 1. 产品概述
- 一句话定位（"为 XX 用户提供 XX 价值的 YY 产品"）
- 产品形态（Web / 移动 / API / 硬件）
- 当前成熟度（MVP / 正式版 / 已商用）

### 2. 核心价值
- 用户痛点 → 产品解决方案
- 量化收益（节省时间 / 降低成本 / 提升效率）
- 情感价值（专业感 / 安全感 / 归属感）

### 3. 目标用户
- 用户画像（行业 / 角色 / 规模）
- 决策链路（影响者 / 决策者 / 使用者）
- 优先级排序（核心用户 / 扩展用户）

### 4. 关键功能
- 3-5 个核心功能
- 每个功能用 1-2 句说明价值
- 避免堆砌技术参数

### 5. 差异化优势
- 与竞品对比（直接竞品 / 间接替代）
- 独特壁垒（技术 / 资源 / 网络效应）
- 不夸大、不贬低对手

### 6. 使用场景
- 2-3 个典型场景（行业 + 角色 + 流程）
- 每个场景配 1 句价值总结

### 7. 定价与获取
- 定价模式（订阅 / 按量 / 一次性）
- 免费试用 / PoC 政策
- 联系方式

### 8. 路线图
- 未来 6-12 个月计划
- 已规划的关键能力
- 邀请反馈的渠道
