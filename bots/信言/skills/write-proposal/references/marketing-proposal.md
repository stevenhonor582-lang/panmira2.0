# 营销方案模板

> 适用 topic_category: 营销方案
> 适用 audience: 客户 / 管理层 / 合作伙伴
> 默认 length: 1500字-3000字

## 必含章节

1. 营销目标 (Marketing Goals)
2. 目标受众 (Target Audience)
3. 核心策略 (Core Strategy)
4. 渠道与执行 (Channels & Tactics)
5. 预算分配 (Budget Allocation)
6. 关键指标 (KPIs)
7. 时间表 (Timeline)
8. 风险与应对 (Risks & Contingency)

## 章节写作要点

### 1. 营销目标
- 业务目标（拉新 / 促活 / 留存 / 品牌曝光）
- 量化指标（如：3 个月内新增 10000 付费用户）
- 与公司 OKR 的对齐

### 2. 目标受众
- 用户画像（行业 / 角色 / 痛点 / 触点）
- 决策链路（影响者 / 决策者 / 使用者）
- 优先级排序（核心 / 扩展 / 培育）

### 3. 核心策略
- 价值主张（用户为什么选择我们）
- 差异化打法（避免同质化价格战）
- 创意主线（一句可传播的话）

### 4. 渠道与执行
- 主渠道（私域 / 公域 / 线下）
- 内容形式（图文 / 短视频 / 直播 / 活动）
- KOL / KOC 合作策略

### 5. 预算分配
- 总盘子与分项（媒介 / 内容 / 活动 / 工具）
- 人力成本与外部采购比例
- 预算上限与审批节奏

### 6. 关键指标
- 过程指标（CPM / CPC / CTR / 转化率）
- 结果指标（GMV / 留存 / NPS）
- 复盘机制（周报 / 月报 / 季度复盘）

### 7. 时间表
- 节奏（预热期 / 爆发期 / 长尾期）
- 关键节点（产品发布 / 大促 / 行业大会）
- 负责人与交付物

### 8. 风险与应对
- 渠道风险（平台政策 / 流量波动）
- 创意风险（用户接受度 / 舆情）
- 预算超支的兜底机制
